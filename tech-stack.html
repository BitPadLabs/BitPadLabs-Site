Our Tech Stack

The tools and technologies we use to build great software

At BitPad Labs, we carefully select technologies that allow us to build scalable, maintainable, and performant software. Our stack choices prioritize developer productivity, code quality, and user experience.

Frontend

React

For building interactive user interfaces with a component-based approach.

Vue.js

For progressive web applications and flexible integration with existing projects.

Tailwind CSS

Utility-first CSS framework for rapidly building custom designs.

TypeScript

Static typing for JavaScript to improve developer experience and code quality.

Backend

Python

Our primary language for backend services, data processing, and automation.

Node.js

For JavaScript/TypeScript backend services and real-time applications.

FastAPI

Modern, fast Python framework for building APIs with automatic documentation.

PostgreSQL

Our database of choice for relational data and complex queries.

Mobile

Swift

Apple's language for iOS, macOS, and watchOS application development.

React Native

For cross-platform mobile development with native performance.

Flutter

Google's UI toolkit for building natively compiled applications.

Kotlin

Modern language for Android development with enhanced safety features.

DevOps & Infrastructure

GitHub Actions

CI/CD pipelines integrated directly with our GitHub repositories.

Docker

Containerization for consistent development and deployment environments.

AWS

Cloud services for hosting, storage, and serverless functions when needed.

Netlify & Vercel

Streamlined hosting and deployment for frontend applications.

AI & Machine Learning

OpenAI APIs

Integration of GPT models for natural language processing capabilities.

TensorFlow

Open-source platform for machine learning and neural networks.

LangChain

Framework for developing applications powered by language models.

Weaviate

Vector database for storing and searching embeddings and semantic data.

We're always exploring new tools and frameworks to improve our development process. If you have suggestions or want to discuss our tech choices, we'd love to hear from you!
